== Tally Framework ==
Computer Repair WordPress Theme is perfect for Computer Repair , cleaning-service, bookkeeping, computer-repair , contractor, dentist and business website. Also It can be use for any type of website. The theme is responsive  that looks great on any device.  This is a child theme of Tally Framework theme and the option panel is powered by Option Tree plugin. It also support TallyKit plugin that will supply lot of functionality.
